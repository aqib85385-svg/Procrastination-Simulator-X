## 🎯 About The Project

**Procrastination Simulator X** is a deliberately useless web app built for fun.

It simulates the feeling of being “almost productive” — with:
- Endless loading screens  
- Fake AI advice  
- Panic moments  
- And completely unnecessary chaos  

This project was created as part of the **DEV April Fools Challenge**.

---

## 😈 Features

- ⏳ **Infinite Loading Bar**  
  Never reaches 100%… just like your motivation  

- 🤖 **Fake AI Assistant**  
  Gives confidently useless advice  

- 🚨 **Boss Approaching Mode**  
  Instantly switches to a fake “work dashboard”  

- 💀 **Virus Mode**  
  Fake system hack with countdown (don’t panic 😄)  

- ☕ **HTTP 418 – I’m a Teapot**  
  A tribute to the most iconic useless status code
